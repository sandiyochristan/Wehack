# We-Hack

A Automated Web Hacking Frame Work for CappricioSec Sec Team


## Install

```bash
# Install We-Hack
cd ~/tools/
git clone https://github.com/sandiyochristan/we-hack.git
cd we-hack/
chmod +x *
./install.sh
```
